# Changelog

## Features

No new features this month. Check back next month for some big new things!

## Small Improvements

- All items in the command bar now match the icons and order of the items in the left-side navigation. <PullRequestLink number="13265" />

  <Author id="luke" />

- The "Generate" and "Disable TOTP" actions now appear as buttons instead of plain text. <PullRequestLink number="13381" />

  <Author id="luke" />

- The agreement signing page for teen applications to HCB is now clearer about the application process. <PullRequestLink number="13456" />

  <Author id="samuel" />

- The "Have you applied for fiscal sponsorship with HCB in the past?" question in the HCB application is now prefilled from our data. <PullRequestLink number="13457" />

  <Author id="samuel" />

- The team member invite modal now specifies that members can view transactions and finances. <PullRequestLink number="13522" />

  <Author id="manu" />

- The application review page is now clearer about why an application may not be ready to submit. <PullRequestLink number="13542" />

  <Author id="samuel" />

- CSV transaction exports now include the amount in both cents and dollars. <PullRequestLink number="13592" />

  <Author id="anish" />

## Bugfixes

- Pressing "Sign out everywhere else" now only updates currently signed-in sessions.
  Previously, it would incorrectly update the timestamp on already-expired sessions. <PullRequestLink number="13375" />

  <Author id="luke" />

- The donations one-pager is now hidden whenever any donations exist, one-time or recurring.
  Previously, if you only had recurring donations, the one-pager would still appear and you couldn't view them. <PullRequestLink number="13384" />

  <Author id="luke" />

- Card names are now shown on the [My cards](https://hcb.hackclub.com/my/cards) page.
  Previously, only card grant purposes were visible here. <PullRequestLink number="13410" />

  <Author id="luke" />

- The donation goal reached and invite accepted emails are now clearer about who can create an announcement. <PullRequestLink number="13444" />

  <Author id="samuel" />

- Hover contrast on orange buttons in dark mode is now fixed. <PullRequestLink number="13523" />

  <Author id="manu" />

- If all of your applications have been archived, HCB will now redirect you to the new application page instead of an archived one. <PullRequestLink number="13536" />

  <Author id="samuel" />
