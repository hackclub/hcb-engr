You can now configure multiple payout settings for reimbursements! This means that you can set up different payout methods such as ACH, Wire, Check, or Wise transfers at the same time.

This allows you to have more flexibility and control over how each of your reimbursements is paid out. If you already have a payout method set up, you will continue to see that as the default option for all new reimbursements. Payout settings for reimbursements that are processing can't be changed until processing is complete.
